# Indeed-Job-Scraper
Web Scraper for 2ยบ project 'Indeed Job' (https://github.com/VitorAmrm/Web-Scraping-Projects);

script was made in perl

to run you need a perl intepreter like [Strawberry perl](https://strawberryperl.com/)

and some Libs.

- WWW::Mechanize
- DBI
- Web::Scraper

just run ```perl script/indeed_scraper.pl```
